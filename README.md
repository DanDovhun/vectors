# vectors

## Setup
### Setup 'gcc' Compiler
Follow instructions on https://developer.fyne.io/started/

### Build
Run command 
> make build

### Clean Build (if 'go.mod' file is missing)
Run command 
>make buildClean

### Run
Run only:
> make run

Build and run:
> make buildAndRun

Clean build and run (if 'go.mod' is missing):
> make cleanBuildAndRun
